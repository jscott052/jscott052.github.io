M6 Milestone #2

StrumAlong

Website Goal

The goal of this website is to provide beginning guitar players with a place to start learning how to play their instrument. It will also have general information about different equipment, especially for electric guitarists such as; Pedals, Multi-Effects, Amplifiers etc.

Working Titles

Home Page

This page will serve as the main navigational area for the website. Each page will still have a navigation bar, however the home page will have that information more readily displayed. This page could feature an image slideshow that links to other pages, and an introductory paragraph.

Tutorial Page

This page will contain a couple of tutorials for beginner guitarists to use. It will use some text and image graphics to help you learn how to play. There will also be a YouTube video tutorial linked on the page.

Equipment Info Page

This page will have information about the different types of equipment that may be required to play guitar. If you're on an electric guitar, there are lots of types of equipment you may need to start playing. There will be information on what kinds of equipment you'd probably want to buy depending on which genre of music you prefer to play

E-Mail Form Page

On this page, visitors will be able to use an E-Mail form to send emails if they have any additional questions about guitar playing.
